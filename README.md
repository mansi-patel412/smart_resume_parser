# 📄 Smart Resume Parser & Job Matcher

## 🚀 Project Overview

The **Smart Resume Parser & Job Matcher** is a web-based application that analyzes resumes and matches them with job roles based on skills. It helps recruiters quickly evaluate candidates and rank them efficiently.

This project is built using **Python and Streamlit**, focusing on automation, data analysis, and user-friendly UI.

---

## 🎯 Features

* 📂 Upload single or multiple resumes (PDF)
* 🛠 Automatic **skill extraction**
* 🎯 **Job role matching**
* 📊 **Match score calculation**
* ❌ Identify **missing skills**
* 🧠 Generate **recruiter insights**
* 🏆 **Candidate ranking system**
* 🥇 Highlight **Top Candidate**
* 📈 **Data visualization (Donut Chart)**
* 🎨 Clean UI with **Light, Dark themes**

---

## 🖥️ Tech Stack

* **Frontend**: Streamlit
* **Backend**: Python

---

## 📊 How It Works

1. Upload resume(s) in PDF format
2. Select a job role
3. System extracts skills from resume
4. Compares with required job skills
5. Calculates:

   * Match Score
   * Matched Skills
   * Missing Skills
6. Displays:

   * Recruiter Insights
   * Candidate Ranking
   * Top Candidate
   * Visualization
